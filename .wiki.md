# Project Summary
The Maple Motion project is a web application tailored for a transportation company in Ottawa, offering services such as corporate shuttles, wedding transfers, and school transportation. The platform features a user-friendly interface for booking services, exploring fleet options, and accessing company values. Users can inquire through a Supabase-powered contact form and utilize a "Demo Mode" for a seamless experience during backend downtimes. The **Articles** feature allows users to engage with content related to transportation and local excursions.

# Project Module Description
- **Home Page**: Introduces the company with a hero image, tagline, and booking call-to-action.
- **Services Page**: Lists available services with descriptions and booking options.
- **Fleet Page**: Showcases vehicles, including a "School Van" instead of a "School Bus".
- **Articles Page**: Users can read and write articles featuring updated content about Ottawa excursions.
- **About Us Page**: Provides background information about the company and its core values.
- **Contact Page**: Contains a form for inquiries with updated contact information.

# Technology Stack
- **React**: Frontend framework for building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Supabase**: Backend service for managing database and authentication.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development and production builds.

# Usage
1. **Install Dependencies**: Run `pnpm install` to install project dependencies.
2. **Build the Project**: Use `pnpm run build` to create a production build.
3. **Run the Development Server**: Execute `pnpm run dev` to start the development server.
